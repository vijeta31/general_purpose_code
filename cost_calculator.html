Pricing Assumptions:
Claude Opus 4.6 via Azure AI Foundry: $5 / 1M input tokens, $25 / 1M output tokens (standard ≤200K context).

LangGraph hosting estimated at ~$15/mo for a lightweight container (Azure Container Apps / App Service B1).
Storage assumed at ~5 GB (vector embeddings + chat history).
All estimates exclude network egress, logging, and monitoring costs.
